Це елемент зазвичай червоного кольору через селектор за назвою елемента.

Цей елемент буде синього кольору через селектор за id.

Цей елемент буде зеленого кольору через селектор за класом.

Цей елемент буде помаранчевого кольору через селектор за кількома класами.

  • Елемент списку 1, фіолетового кольору через селектор за вкладеністю.
  • Елемент списку 2, також фіолетового кольору через селектор за вкладеністю.
